## 🚀 Proyectos destacados

- **[NeuroLinked](https://neurolinked.vercel.app)** – Modelo de IA que interpreta señales EEG reales para accionar domótica y devolver autonomía a personas con movilidad reducida.  
  Permite a cualquiera controlar domótica de forma innovadora y accesible.  
  *Tecnologías:* IA aplicada, domótica, EEG, accesibilidad.  
  [Código](https://github.com/emitejadaa/NeuroLinked)

- **[Secure Track](https://secure-track.vercel.app)** – Plataforma backend + hardware para administrar carritos de computadoras compartidas con monitoreo, cierres seguros y panel web.  
  Reduce pérdidas de equipos y facilita la logística tecnológica en instituciones educativas y empresas.  
  *Tecnologías:* Backend, IoT, Flask, Raspberry Pi.  
  [Código](https://github.com/3-TIC-ORT/proyecto-3-secure-track)

- **[CSBS-DP Web Form](https://csbs-form.vercel.app)** – Aplicación web para digitalizar y puntuar automáticamente el cuestionario clínico CSBS-DP, enviando un reporte estructurado por email al profesional de la salud.                                                                                                  
  *Tecnologías:* Next.js, TypeScript, Serverless, Email Automation.                                                                                            
  [Código](https://github.com/emitejadaa/csbs-form)
